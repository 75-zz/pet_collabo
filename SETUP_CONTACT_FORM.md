# コンタクトフォームのセットアップ手順

このプロジェクトでは、Web3Formsを使用してコンタクトフォームからのメール送信を実装しています。

## セットアップ手順

### 1. Web3Formsのアカウント作成

1. [Web3Forms](https://web3forms.com) にアクセス
2. 「Get Started for Free」をクリック
3. メールアドレスを入力してアカウント作成
4. メール認証を完了

### 2. アクセスキーの取得

1. Web3Formsのダッシュボードにログイン
2. 「Access Keys」セクションでアクセスキーを確認
3. アクセスキーをコピー

### 3. 環境変数の設定

1. プロジェクトルートに `.env` ファイルを作成（`.env.example` を参考）
2. アクセスキーを設定:

```bash
VITE_WEB3FORMS_ACCESS_KEY=あなたのアクセスキー
```

### 4. コードの更新

`src/app/components/ContactSection.tsx` の70行目を以下のように変更:

```typescript
// 変更前
const accessKey = 'YOUR_WEB3FORMS_ACCESS_KEY';

// 変更後
const accessKey = import.meta.env.VITE_WEB3FORMS_ACCESS_KEY;
```

### 5. 開発サーバーの再起動

環境変数を読み込むため、開発サーバーを再起動:

```bash
npm run dev
```

## テスト

1. ブラウザで http://localhost:5173 にアクセス
2. 「Get in Touch」セクションまでスクロール
3. フォームに情報を入力して送信
4. 成功メッセージが表示されることを確認
5. Web3Formsのダッシュボードで送信が記録されていることを確認

## 送信先メールアドレスの設定

Web3Formsのダッシュボードで、フォーム送信の通知を受け取るメールアドレスを設定できます。

## 無料プランの制限

- 月間250件まで無料
- それ以上必要な場合は有料プランへのアップグレードが必要

## トラブルシューティング

### フォームが送信されない

1. ブラウザのコンソールでエラーを確認
2. アクセスキーが正しく設定されているか確認
3. 環境変数が `.env` に正しく記載されているか確認
4. 開発サーバーを再起動したか確認

### エラーメッセージが表示される

- ネットワーク接続を確認
- Web3Formsのサービスステータスを確認
- アクセスキーの有効性を確認
